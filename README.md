# Hospital Analytics Dashboard

## Project Overview

The Hospital Analytics Dashboard is developed using Power BI to analyze hospital data and provide meaningful insights into patient care, hospital operations, and financial performance. The dashboard helps in monitoring key performance indicators (KPIs) such as patient admissions, bed occupancy, treatment costs, and departmental performance.

## Tools & Technologies

* Power BI
* Excel
* Data Modeling
* DAX

## Key Features

* Interactive dashboards for hospital performance analysis
* Patient admission and discharge analysis
* Department-wise performance tracking
* Financial and operational KPI monitoring
* Data visualization for better decision-making

## Purpose

The main goal of this project is to transform raw hospital data into actionable insights using data visualization and analytics, helping healthcare management improve efficiency and decision-making.
